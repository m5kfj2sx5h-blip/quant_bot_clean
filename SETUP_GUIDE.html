Quant Bot Setup Guide

Complete setup and execution instructions for the arbitrage trading system.

1 Prerequisites

Required Software

  • Python 3.9+ installed
  • pip package manager
  • Git (optional, for updates)

Required Accounts

  • Binance.US - With API keys (trading enabled)
  • Kraken - With API keys (Kraken Pro subscription recommended)
  • Coinbase - Standard API keys
  • Coinbase Advanced - Advanced Trade API keys (separate from standard)
  • TradingView - Pro subscription for webhook alerts

2 Installation

# Navigate to the bot directory
cd quant_bot_final

# Install dependencies
pip install -r requirements.txt

# Or install manually:
pip install ccxt python-dotenv psutil aiohttp websockets apscheduler streamlit plotly pandas

4 TradingView Webhook Setup

Webhook URL

http://YOUR_SERVER_IP:8090/signal

MACRO Signal Format (BTC/GOLD Mode)

{
    "secret": "your_tv_webhook_secret",
    "type": "MACRO",
    "action": "BTC"
}

Use "action": "GOLD" to switch to GOLD mode.

A-Bot Sniper Signal Format

{
    "secret": "your_tv_webhook_secret",
    "type": "SNIPER",
    "action": "BUY",
    "symbol": "SOL",
    "exchange": "kraken"
}

Use "action": "SELL" to sell. Exchange is optional.

5 Running the Bot

Start the Main Bot

python main.py

Start the Dashboard (separate terminal)

streamlit run dashboard.py

Dashboard will be available at http://localhost:8501

  • 20% - Triangular Arbitrage (30s cycles)

7 Manual Actions

Gold Sweep (Dashboard)

Manual only. Sweeps 15% of PAXG profits to cold wallet. Max 2x per month.

Manual Transfer Notification

When AUTO_TRANSFER_ENABLED=false, the dashboard will alert you when drift exceeds 15% and recommend the best network (Tron/Solana) for manual transfer.

Mode Override (Dashboard)

BTC MODE and GOLD MODE buttons allow manual override of the macro signal.

Important Reminders

  • This is a LONG-ONLY spot trading system. No shorting, margin, or futures.
  • Pre-position capital across all exchanges before starting.
  • G-Bot NEVER gives up more than 85% of its holdings. 15% stays forever.
  • Arbitrage uses pre-positioned capital. No transfers per trade.
  • Test with small amounts first!
